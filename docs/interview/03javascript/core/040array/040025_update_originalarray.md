# 数组的哪些 API 会改变原数组？

## 题干

- 改变原数组的数组 API

## 题解


- `push()`：向数组末尾添加一个或多个元素。

- `pop()`：从数组末尾删除一个元素。

- `shift()`：从数组开头删除一个元素。

- `unshift()`：向数组开头添加一个或多个元素。

- `splice()`：从数组中删除元素，并可以在删除的位置插入新元素。

- `sort()`：对数组进行排序。

- `reverse()`：反转数组元素的顺序。

- `fill()`：用一个固定值填充数组。

- `copyWithin()`：从数组中复制一段元素，并将其插入到另一个位置。
